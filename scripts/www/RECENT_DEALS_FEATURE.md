# 近期单车成交价格功能

## 功能概述

在车型搜索页面添加了**近期单车成交价格展示**模块，显示最近7天或30天的具体成交记录。

## 功能特性

### 1. 时间范围切换
- **近7天**: 显示最近一周的成交数据
- **近30天**: 显示最近一个月的成交数据
- 点击标签即可切换

### 2. 统计指标
- **平均成交价**: 所选时间段的平均价格
- **成交数量**: 总成交车辆数
- **最高成交价**: 最高价
- **最低成交价**: 最低价
- **价格变化**: 与前期相比的涨跌百分比

### 3. 价格趋势图
- 显示每日平均成交价走势
- 面积图展示价格波动
- 悬停显示具体数值

### 4. 成交记录列表
- 显示最近10条成交记录
- 包含：日期、年份、车况、里程、成交价
- 按时间倒序排列

## 数据生成

所有数据均在**本地生成**，无需网络：

```javascript
// 成交记录生成逻辑
{
  date: "02-15",          // 成交日期
  fullDate: "2024-02-15", // 完整日期
  price: 1250000,         // 成交价（基于基础价格±8%浮动）
  mileage: 5.5,           // 里程（1-15万公里随机）
  year: 2021,             // 年份（2019-2024随机）
  condition: "良好",       // 车况（按权重分布）
  location: "曼谷"         // 地点
}
```

### 车况分布权重
- 优秀: 20%
- 良好: 50%
- 一般: 25%
- 较差: 5%

## 使用方式

1. 进入**车型搜索**页面
2. 选择品牌和车型
3. 点击**搜索市场数据**
4. 在结果页面查看**近期单车成交价**模块
5. 点击**近7天**或**近30天**切换时间范围
