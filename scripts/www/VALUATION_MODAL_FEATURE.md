# 估价弹窗功能

## 功能概述

在搜索结果页面点击"立即估价"按钮时，弹出小窗口显示：
1. 选择车型的7天内价格走势图
2. AI推荐售价和建议区间

## 关闭方式

1. 点击右上角 X 按钮
2. 点击弹窗外部的半透明遮罩
3. 按键盘 ESC 键
4. 点击"关闭"按钮

## 技术实现

### 新增文件修改

| 文件 | 修改内容 |
|------|----------|
| `index.html` | 添加估价弹窗HTML结构 |
| `css/dashboard.css` | 添加弹窗样式和估价专用样式 |
| `js/modules/uiManager.js` | 添加弹窗逻辑和图表渲染方法 |

### 主要方法

```javascript
// 绑定估价按钮事件
bindSearchValuationButton()

// 打开估价弹窗
openValuationModal()

// 关闭估价弹窗
closeValuationModal()

// 生成7天价格走势数据
generate7DayTrend(brandId, modelId, countryCode)

// 渲染弹窗结果
renderValuationModalResult(data)

// 渲染7天走势图
renderValuationTrendChart(data)
```

## 数据说明

- **7天走势图**: 基于车型基础价格生成，带有±2%的随机波动模拟真实市场
- **AI推荐价格**: 基础价格的95%~105%随机值
- **建议区间**: 推荐价格的±8%
- **AI置信度**: 固定显示92%（模拟值）

## 使用流程

1. 在搜索页面选择品牌和车型
2. 点击"搜索市场数据"
3. 在AI智能建议区域点击"立即估价"按钮
4. 弹窗显示7天价格走势和AI推荐价格
5. 可选择"关闭"或跳转到"详细估价"页面
