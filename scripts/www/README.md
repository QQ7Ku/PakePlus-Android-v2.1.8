# 新能源二手车3D可视化检测系统

基于 Three.js 的 BYD Qin Pro DM 2019 3D检测平台

## 功能特性

### 1. 3D模型展示
- 使用 Three.js 构建的交互式3D场景
- 内置 BYD Qin Pro DM 2019 程序化生成模型
- 支持外接FBX模型加载
- 多种视图切换（前/后/左/右/俯视/等轴测）

### 2. 漆面检测热点
- 18个标准漆面检测点：
  - 前部：引擎盖、前保险杠、左/右前翼子板
  - 侧面：左/右前门、左/右后门、左/右后翼子板
  - 后部：后备箱盖、后保险杠
  - 顶部：车顶
  - 立柱：A柱/B柱/C柱（左/右）
- 可视化热点标记
- 支持显示/隐藏特定检测点

### 3. 问题管理
- 添加检测问题（划痕、凹陷、褪色、剥落等）
- 严重程度分级（轻微/中等/严重）
- 图片上传功能
- 修复建议与费用预估

### 4. 报告导出
- **Word报告**: 生成标准格式的检测报告(.doc)
- **PDF报告**: 生成PDF格式的检测报告
- **数据导出**: 导出JSON格式检测数据

### 5. 商家/顾客双模式
- 顾客模式：查看检测结果
- 商家模式：编辑检测数据、添加问题

### 6. 数据管理
- 保存/加载检测数据
- 数据持久化（LocalStorage）
- 重置默认数据

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 等待3D模型加载完成
3. 点击3D模型上的热点查看检测点详情
4. 在商家模式下可以添加检测问题
5. 导出Word或PDF报告

## 技术栈

- **Three.js** - 3D图形库
- **GSAP** - 动画库
- **jsPDF** - PDF生成
- **Font Awesome** - 图标
- **原生JavaScript** - 无框架依赖
