# EV Export Pro - 新能源汽车出海智能助手

一个专为新能源汽车二手车出口到东盟国家设计的智能演示系统。该系统提供AI证件识别、检测报告分析、人工审核、报关信息确认和政策合规检测等全流程服务。

## 🌟 主要功能

### 1. AI行驶证识别 (Step 1)
- 上传行驶证照片，AI自动识别车辆信息
- 支持拖拽上传和点击上传
- 自动提取车牌、VIN码、品牌型号等关键信息

### 2. 检测报告智能分析 (Step 2) ⭐增强
- 支持上传第三方检测报告图片
- AI自动分析检测项目并提取关键数据
- **检测项目明细展示** - 按类别展示所有检测项目
- **检测项目编辑** - 点击任意项目可修改名称、状态、备注
- 分类包括：车身漆面(13项)、基本照片(26项)、车身骨架(16项)、必检项(11项)、车况检测(6项)、机电系统(9项)

### 3. 人工审核 - 检测报告编辑 (Step 3)
- **可编辑的检测项目列表**
  - 支持添加、删除、修改检测项目
  - 每个项目包含：标题、描述、严重程度、类别、修复费用
- **审核信息录入**
  - 审核人姓名
  - 审核日期
  - 审核备注
- **检测评分调整**
  - 综合评分
  - 检测等级
  - 正常/需注意/异常项目数量

### 4. 车辆及报关信息确认 (Step 4) ⭐增强
完整的报关确认单信息：

#### 车辆基本信息
- 车牌、品牌、型号、VIN码
- 发动机号、注册日期、行驶里程
- 车辆颜色、使用性质、动力源类型

#### 新能源信息
- 电池类型、容量、健康度、续航里程
- 电池状态、是否属危险品、包装方式

#### 出口报关信息 ⭐
- 报关日期、经营企业名称
- 统一社会信用代码

#### 出口信息 ⭐
- 出口方式（一般贸易/边境小额）
- 出口口岸、发票号码、合同号
- 成交方式（FOB/CIF/CFR）、目的港及国别

#### 海关申报信息 ⭐
- 商品编号（HS CODE，默认8703.80）
- 申报口岸、进境关别
- 运输方式、运输工具名称

#### 证书信息 ⭐
- 机动车登记证书编号、原车牌号码
- 出口许可证号、商检单号

#### 检测结果摘要
- 显示已审核的检测数据
- 检测项目列表预览

### 5. 目标国家选择 (Step 5)
- **重点支持国家**：柬埔寨 🇰🇭、泰国 🇹🇭
- 支持全部10个东盟成员国
- 智能搜索和区域筛选
- 高亮显示推荐出口国家

### 6. AI政策合规检测 (Step 6)
- 自动分析车辆是否符合目标国家进口政策
- 检测项目：车龄限制、电池健康度、驾驶位方向、进口关税、排放标准
- **导出Word报告** - 完整的出海检车报告（.doc格式）
- **导出文本报告** - 合规检测结果
- **导出报关确认单** - 完整的报关信息汇总单

## 🔄 步骤导航 ⭐新增功能

### 点击步骤跳转
点击顶部步骤指示器（1-5）可直接跳转到对应步骤：
- **Step 1** - 证件识别
- **Step 2** - 报告识别（含检测项目明细）
- **Step 3** - 人工审核
- **Step 4** - 报关信息
- **Step 5** - 目标国家
- **Step 6** - 政策检测

### 前置条件检查
- 必须完成前一步骤才能进入下一步
- 步骤指示器会显示已完成、当前、待完成状态

## 📋 报告导出功能

### Word报告导出 ⭐新增
导出专业的Word文档报告（.doc格式），包含：

1. **封面** - 报告标题、编号、生成时间
2. **车辆基本信息** - 车牌、品牌、型号、VIN等
3. **新能源信息** - 电池类型、容量、健康度、续航
4. **检测结果** - 综合评分、检测问题列表、审核信息
5. **目标国家信息** - 出口目的地
6. **合规检测结果** - 详细检测项、出口相关信息
7. **报关信息摘要** - 经营企业、出口方式、HS编码等
8. **页脚** - 生成时间和免责声明

报告采用专业排版，带有表格、颜色标识和评分可视化。

### 报关确认单导出
根据标准二手车新能源汽车出口报关信息汇总确认单格式，导出包含：

1. **报关信息** - 日期、企业名称、信用代码
2. **车辆信息** - 品牌/VIN/注册日期/里程/颜色等
3. **出口信息** - 方式/口岸/合同/成交方式/目的港
4. **海关信息** - HS CODE/申报口岸/运输方式
5. **证书信息** - 登记证书/许可证号/商检单号
6. **电池信息** - 容量/续航/状态/危险品标识
7. **检测结果** - 机构/报告号/评分/项目列表
8. **审核信息** - 审核人/日期/备注

## 🚀 使用方法

### 快速开始
1. 打开 `index.html` 文件
2. 点击"使用演示数据"快速体验完整流程

### 完整流程
1. **证件识别** - 上传或演示数据
2. **报告识别** - 查看检测项目明细，点击项目可编辑
3. **人工审核** - 编辑检测项目，填写审核信息
4. **报关信息** - 填写完整报关信息
5. **目标国家** - 选择出口国家
6. **政策检测** - 查看合规结果，导出Word报告/文本报告/报关确认单
