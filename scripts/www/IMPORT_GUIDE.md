# 东盟二手车1000问批量导入指南

## 使用方法

### 方法一：使用批量导入工具（推荐）

1. 打开 `batch-import.html` 页面
2. 将您的1000条Q/A内容复制粘贴到输入框
3. 选择分类（或选择"自动识别"）
4. 点击"开始导入"
5. 等待导入完成（约1-2分钟）

### 方法二：使用智能提取（单条/少量）

1. 打开 `admin.html` 管理界面
2. 点击"智能提取"按钮
3. 粘贴Q/A例句，格式：
   ```
   Q: 客户加微信后怎么跟进？
   A: 首次发送：①车辆实拍视频...
   ```
4. 系统自动提取关键词、分类等信息
5. 确认后添加到知识库

## Q/A格式要求

支持的格式：
```
Q1: 东盟二手车市场主要包括哪些国家？
A1: 核心市场包括泰国、马来西亚、印度尼西亚...

Q2: 东盟二手车交易的主要车型有哪些？
A2: 日系车（丰田、本田、日产）占主导地位...
```

或简写格式：
```
Q: 问题内容
A: 答案内容
```

## 批量导入步骤

1. **分批导入**：建议每批导入50-100条，避免浏览器卡顿
2. **检查日志**：导入后查看成功/失败记录
3. **导出备份**：导入完成后导出JSON备份

## 注意事项

1. 浏览器本地存储有限（约5-10MB），大量数据建议使用外部JSON文件
2. 导入过程中请勿关闭页面
3. 导入失败通常是因为内容重复或格式问题
4. 建议先导入一小批测试，确认无误后再大批量导入
